User Administration - User Summary can be accessed as follows:
  1. Click the View User button on User Administration - Choose User.

User Administration - User Summary allows you to view the security record details of a Client-Central user associated with your institution.

User Administration - User Summary contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user.
  • Last Name – Last name of the user.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.

User Details (contains view-only fields)

  • First Name – First name of the user.
  • Last Name – Last name of the user.
  • Logo – Four-character alphanumeric client identifier.
  • Group/User – Numeric identifier that labels the organization and user for the host computer.
  • Audit Rpt. Institution – Client ID of the insitution for which a log of the user activity is generated.
  • Networks Only – Indicates whether network record access is available to this user. This field is reserved for network use.
  • Institution – Name of the institution.
  • Address – Address of the user.
  • Country – Country of the user.
  • City – City of the user.
  • State – State of the user.
  • ZIP – ZIP code of the user.
  • Phone – Phone number of the user.
  • Ext – Extension of the user.
  • Fax – Fax number of the user.
  • E-mail – E-mail address of the user.
  • Date Created – Date on which the user record was created.
  • Date Last Login – Date on which the user last logged on to Client-Central.
  • Date Last Modified – Date on which the user record was last modified.

Token Information (contains view-only fields)

  • Token Serial Number – Serial number of the token allocated to the user.
  • Action – Token request type. This field displays only when requesting a new or replacement token.
      Shipping Address
  • Institution – Name of the institution to which the token should be shipped.
  • Deliver To – Name of the person designated to receive the token.
  • Address – Destination address of the shipment.
  • Country – Destination country of the shipment.
  • City – Destination city of the shipment.
  • State – Destination state of the shipment.
  • ZIP – Destination ZIP code of the shipment.
  • Phone – Phone number of the person designated to receive the token.
  • Ext – Extension of the person designated to receive the token.
  • Fax – Fax number of the person designated to receive the token.
  • Email – E-mail address of the person designated to receive the token.
Note: The Shipping Address section displays only when requesting a new or replacement token.

Applications (contains view-only fields)

Applications, files, and access rights as defined on the security record of the user.

Special Roles (contains view-only fields)

Special roles applicable to the user.

Logos (contains view-only fields)

Logos associated with the user.

Availability (contains view-only fields)

Time restrictions applicable to the user.

Buttons, links, and icons contained in User Administration - User Summary:

Buttons
Manage User – Allows you to update or delete the user record.

Add New User – Allows you to add new user records.

Links
Print – Prints the user record.

Icons
None
