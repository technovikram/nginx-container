User Administration—Add User Record

The last step in the add process is to save the user security record. Before you select Save on the Add User Record page, the new record can still be cancelled if you click the Cancel button. It is not added until you check the Approval box and click Save. This navigation page gives you the opportunity to go back and double-check or correct any of the information previously entered in the Add Wizard before you save the record. You can do so by selecting the relevant tab—Details, Institution IDs, Applications, Special Roles, Availability, and Tokens—review or make your change, and click the OK button to return to this page.

Note: The Access Rights page is accessible from within the Applications page. After you have provided all relevant information for the new user, check the Approval box and click Save. A confirmation page displays with the new user ID. See the Security manual for further details.
