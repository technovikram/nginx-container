User Administration - Navigation can be accessed as follows:
  1. Click the Continue button on User Administration - Tokens.

    2. - or -
  2. Click the Manage User button on User Administration - Choose User.

User Administration - Navigation allows you to add or modify the user record on Client-Central, based on the information you have supplied.

You may access the following pages via the User Record tab within User Administration - Navigation:

  • User Administration - Details.
  • User Administration - Logos.
  • User Administration - Applications.
  • User Administration - Special Roles.
  • User Administration - Accessibility.
  • User Administration - Tokens.

Note: User Administration - Special Roles is visible only if applicable.

User Administration - Navigation contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user whose record is being accessed or created.
  • Last Name – Last name of the user whose record is being accessed or created.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.
Note: The user ID displays only when an existing user record is being accessed.

Approval (contains modify fields)

Prior to saving the record you must accept the statement. For further information contact your client services representative.

Buttons, links, and icons contained in User Administration - Navigation:

Buttons
Save – Creates a new user record and initiates the processing of the requested administrative tasks.

Cancel – Returns you to User Administration - Choose User without saving changes.

Update – Updates the user record and initiates the processing of the requested administrative tasks.

Delete – Deletes the user record.

Note: The Update and Delete buttons display only when accessing an existing user record.
