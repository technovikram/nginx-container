User Administration - Applications

You can control access to a number of applications via the User Administration application. This page allows you to grant or revoke user access to applications. You can select and move one or more of these applications between the Available Applications list and the Selected Applications list. The user has access to the applications within the Selected Applications list.

Note: Every institution has one designated primary security administrator (PSA). Maximum privileges applicable to the institution are defined on this user's security record. The PSA can give other administrators and other users the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please refer to the EFT User Manual, User Administration for details.

Applications
Allows you to control the user's applications access.
Available Applications Applications available for access. This list is limited to the applications specified on your own user record. You can select relevant applications from this list and move them to the Selected Applications list using the buttons provided.

Note: EFT Security Administrators have tasks that may take up to 5 business days for the applications marked with * (asterisk). The status of these tasks can be tracked on the Service Request Report.

Selected Applications The user is granted access to all the applications within the Selected Applications list.

Buttons
>> Moves the selected (one or more) applications from the Available Applications list to the Selected Applications list.
All >> Moves all applications from the Available Applications list to the Selected Applications list.
<< Moves the selected (one or more) applications from the Selected Applications list to the Available Applications list.
All << Moves all applications from the Selected Applications list to the Available Applications list.
Continue Temporarily stores the information entered on this page and takes you to the Access Rights page where you can establish or modify rights for new or existing applications.
Cancel Discards your changes and returns you to the User Record page.
