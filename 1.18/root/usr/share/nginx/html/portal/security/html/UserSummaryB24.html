User Administration—User Summary

User Summary allows you to view security record details for all users associated with your institution. It gives you a comprehensive view of the user record and an opportunity to print the page. Changes cannot be made on this page, but you can select Manage User if updates to the user record are required.

  • Key Information—Displays the User Name, User ID, and Primary Institution ID of the selected user record.
  • User Details—Displays relevant user details: System, User Name, Address, User IDs, and relevant maintenance details.
  • Token Information—Displays all hard and/or soft tokens assigned to the user record.
  • Applications—Displays all Applications, Permissions, and Access Rights assigned to the user.
  • Special Roles—Displays all special roles assigned to the user.
  • OrgIDs—Lists all OrgIDs available to the user.
  • FIIDs—Lists all FIIDs available to the user.
  • Availability—Displays the time period (Pacific Time) the user is allowed to access Client-Central.
  • Manage User button—Takes you to the User Record page where you can update, disable/enable, or delete the user record.
  • Print link—Allows you to print the page.

All relevant sections display based on your institution's parameter setup and your access rights. For more information about a specific field, go to the page help of the associated page.
