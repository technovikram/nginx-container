User Administration - Add Availability

Access to Client-Central can be further controlled by time. This page allows you to specify time restrictions for the user. The user is granted access only during the allotted time frame (U.S. Eastern Time).

Availability (Eastern Time)
Allows access time to be specified on the new user's security record.
Start Time Time of day when the user can begin to access Client-Central. Time format is military time and is defaulted to 00:00.
End Time Time of day after which the user can no longer access Client-Central. Time format is military time and is defaulted to 23:59.

Buttons
Back The Back button temporarily stores the access times specified on the new user's security record and takes you to the previous page.

Clicking Back from the Add Availability page takes you to the Add Special Roles page.

Continue The Continue button temporarily stores the access times specified on the new user's security record and takes you to the next page.

Clicking Continue from the Add Availability page takes you to the Add Token page.

Cancel The Cancel button discontinues the add user process and returns you to the Choose User page.

Clicking Cancel from the Add Applications page displays the following confirmation message:
To cancel this Add, click OK.
To return to the Add, click Cancel.
