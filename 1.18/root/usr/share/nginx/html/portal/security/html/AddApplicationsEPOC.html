User Administration - Add Applications

You can control access to a number of applications via the User Administration application. The Add Applications page allows you to grant new users access to the specific applications you want them to use. All available applications display on the page. You can select and move applications between the two lists using the buttons provided. The new user will have access to the applications in the Selected Applications list.

Note: Every institution has one designated primary security administrator (PSA). Maximum privileges applicable to the institution are defined on this user's security record. The PSA can give other administrators and other users the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please refer to the EFT User Manual for User Administration details.

Applications
Allows you to specify applications access on the new user's security record.
Available Applications Applications available for access. This list is limited to the applications specified on your own user record. You can select relevant applications from this list and move them to the Selected Applications list using the buttons provided.

Note: EFT Security Administrators have tasks that may take up to 5 business days for the applications marked with * (asterisk). The status of these tasks can be tracked on the Service Request Report.

Selected Applications The user is granted access to all the applications within the Selected Applications list.

Buttons
>> Moves the selected (one or more) applications from the Available Applications list to the Selected Applications list.
All >> Moves all applications from the Available Applications list to the Selected Applications list.
<< Moves the selected (one or more) applications from the Selected Applications list to the Available Applications list.
All << Moves all applications from the Selected Applications list to the Available Applications list.
Back Takes you to the previous page.
Continue Temporarily stores the set of applications to be specified on the new user's security record and takes you to the Add Access Rights page.
Cancel Discontinues the add user process and returns you to the Choose User page.

Clicking Cancel from the Add Applications page displays the following message: Click OK to exit Add User.
