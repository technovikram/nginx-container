User Administration—Add Institutions

Client-Central provides the user with access to cardholder information relating to one or more client institutions, which are named FIIDs for Debit clients and OrgIDs for Full Service Credit clients. This page allows you to grant the new user access to specific FIIDs and/or OrgIDs.

When the user has the BASE24 System checked on the User Details page, the FIIDs section displays. When the Full Service Credit System is checked, the OrgIDs section displays.

The Add Institutions page displays for all users and shows all of your financial institution IDs. Many users have access to only one institution and if that is the case, that is the only FIID or OrgID that displays on this page. If you have access to multiple institutions, they all display in the Available list and you are allowed to determine which FIIDs and/or OrgIDs the new user should have access to by moving them to the Selected list, using the buttons on the page.

Note: Every institution has one designated primary security administrator. Maximum privileges applicable to the institution are defined on this user's security record. Other administrators and other users can be given the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please see the Security manual for further details.

FIIDs for LNET
Allows you to specify FIIDs that can be accessed by the new user.
Available FIIDs Financial institution IDs available for access. You can highlight FIIDs in this list and move them to the Selected FIIDs list for the new user.

Note: This list is limited to the FIIDs specified on your own user record.

Selected FIIDs

The new user is granted access to all the FIIDs within the Selected FIIDs list.

To remove access from the user record, highlight FIIDs in this list and move them to the Available FIIDs list.
Back to Top

OrgIDs
Allows you to specify OrgIDs that can be accessed by the new user.
Available OrgIDs

OrgIDs available for access. You can highlight OrgIDs in this list and move them to the Selected OrgIDs list.

Note: This list is limited to the OrgIDs specified on your own user record.

Selected OrgIDs

The user is granted access to all the OrgIDs within the Selected OrgIDs list.

To remove access from the user record, you can highlight OrgIDs in this list and move them to the Available OrgIDs list.
