CardValet
Assigned FI Token Drop down list of available FI Tokens from Ondot Database.
AppToken Display Name Drop down list of available AppToken which were mapped on the Manage Institution page and assigned to the Digital Certificate ID.
Create WalletApp User When checked, credentials for a new WalletApp User can be entered.

When not checked, a new user will not be created. Field will not appear in Manage User page.

User ID Identifier assigned by Card Services for the selected user displays when the Add WalletApp User checkbox is checked. Format should be LOGO_WA.

Field is read-only.

Password Enter the desired password for the new AppToken.

Note: Password cannot be less than 8 characters.

Field cannot be edited from this screen after UserID and password have been added. Password must be changed from the Manage User page.

Re-Enter Password Re-enter the desired password.

Client Search Map
Primary Client ID column Displays the primary client ID. This section is used when a Financial Institution has more than one logo and allows a card search across those multiple logos.
Search Client ID(s) column Displays the client ID associated with the primary client ID.

Multiple Client IDs are separated by a comma.

Edit Select to modify the Search Client ID associated with the Primary Client ID.

When selected, a dialog box displays showing all available Client IDs. From here you can move any desired Client IDs to the Search Client IDs column. Select Apply to save changes, or Cancel to close the dialog box.
