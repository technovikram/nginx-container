User Administration - Defaults

A default token shipping address can be specified for your institution. This page allows you to enter details of this address. Please ensure that a relevant fax number is provided. On the tokens page, this address is used when the Default option is selected.

Shipping Address
This section lists the default token shipping address for your institution.
Institution Name of the institution to which the token is being shipped.
Deliver To Name of the person designated to receive the token.
Address Shipping address.
Country Destination country of the shipment.
City Destination city of the shipment.
State Destination state of the shipment.
ZIP Destination ZIP code of the shipment.
Phone Phone number of the person designated to receive the token.
Ext Extension of the person designated to receive the token.
Fax Fax number of the person designated to receive the token.
Email Email address of the person designated to receive the token.
