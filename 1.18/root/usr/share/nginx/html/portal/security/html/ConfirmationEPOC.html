User Administration—Confirmation

This page displays with a confirmation message when you have completed the process of updating the user security record. It gives you a comprehensive view of the user record and an opportunity to print the page. Changes cannot be made on this page but you can select Manage User if further updates to the user record are required.

Each page of the user record has its own section within the Confirmation page as follows:

  • Key Information—Displays the User Name, User ID, and Primary Institution ID of the selected user record.
  • User Details—Displays relevant user details—System, User Name, Address, User IDs, and relevant maintenance details.
  • Token Information—Displays all hard and/or soft tokens assigned to the user record.
  • Applications—Displays all Applications, Permissions, and Access Rights assigned to the user as on the Applications and Access Rights page.
  • Special Roles—Displays all Special Roles assigned to the user as on the Special Roles page.
  • OrgIDs—Lists all OrgIDs available to the user.
  • Logos/Client IDs—Lists all logos available to the user.
  • Availability—Displays the time period (Eastern Time) the user can access Client-Central.
  • Manage User button—Displays the User Record page where you can update, disable/enable, or delete the user record.
  • Print link—Allows you to print the page.
All relevant sections display based on your institution's parameter setup and your access rights.

For more information about a specific field, please go to the page-level help for the associated page.
