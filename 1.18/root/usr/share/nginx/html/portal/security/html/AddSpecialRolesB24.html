User Administration - Add Special Roles

In addition to the normal read, add, delete, and update permissions, you can set up special roles within an application. A special role must be assigned in order for the user to be able to perform that specific task. If a user does not have a special role, even if they have full access to the rest of the permissions for the application, they are not allowed to perform that specific task.

The special roles that display on this page depend on the selected applications given to the user who is being managed, as well as the special roles that you have access to. This page allows you to manage user access to the available special roles. You can select and move one or more of these special roles between the available and the selected lists. The user is granted access to the roles in the Selected Roles list.

Note: Every institution has one designated primary security administrator (PSA). Maximum privileges applicable to the institution are defined on this user's security record. Other administrators and other users can be given the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please refer to the EFT User Manual for User Administration details. `

Special Roles
Allows you to specify special role access for the new user.
Available Roles Special roles available for access depends on the user's selected applications. This page will not display in the add process if the selected applications do not have special roles established.

You can highlight roles on this list and move them to the Selected Roles list using the buttons provided.

Selected Roles The user is granted access to all of the special roles within the Selected Roles list.

Buttons
>> Moves the highlighted roles from the Available Roles list to the Selected Roles list.
All >> Moves all roles from the Available Roles list to the Selected Roles list.
<< Moves highlighted roles from the Selected Roles list back to the Available Roles list.
All << Moves all roles from the Selected Roles list back to the Available Roles list.
Back Takes you to the previous page.

Clicking Back from the Add Special Roles page takes you to the Add Access Rights page.

Continue Temporarily stores the data entered on this page for the new user and takes you to the next page.

Clicking Continue from the Add Special Roles page takes you to the Add Availability page.

Cancel Discontinues the add user process and returns you to the Choose User page.

Clicking Cancel from the Add Special Roles page displays the following confirmation message: Click OK to exit Add User.
