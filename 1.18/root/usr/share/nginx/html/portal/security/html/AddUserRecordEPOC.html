User Administration—Add User Record

The last step in the add process is to save the user security that you have just added. Before doing so, you have the opportunity to review and modify any of the information entered. You can do so by selecting the relevant page from the second row of menu items displayed on the page. The menu items available include Details, Institution IDs, Applications, Special Roles (visible only if applicable), Availability, and Token. Please note that the Access Rights page is accessible from within the Applications menu item.

After ensuring that you have provided all relevant information, check the Approval check box. Refer the Security manual for further details. Select Save to add the new user and display a confirmation page.

Buttons
Save Adds the new user security record to Client-Central, initiates the processing of the requested administrative tasks (if any), and takes you to a confirmation page.

Clicking Save from the Add User Record page takes you to the Confirmation page.

Cancel Discontinues the add user process and returns you to the Choose User page.

Clicking Cancel from the Add Token page displays the following confirmation message:
To cancel this Add, click OK.
To return to the Add, click Cancel.
