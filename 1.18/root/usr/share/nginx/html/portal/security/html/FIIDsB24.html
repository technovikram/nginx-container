User Administration—Institutions

Client-Central provides the user with access to cardholder information relating to one or more client institutions, which are named FIIDs for Debit clients and OrgIDs for Full Service Credit clients. This page allows you to grant or remove user access to specific FIIDs and/or OrgIDs.
When the user has the Base24 System checked on the User Details page, the FIIDs section displays.  When the Full Service Credit System is checked, the OrgIDs section displays.

Note: Every institution has one designated primary security administrator. Maximum privileges applicable to the institution are defined on this user's security record. Other administrators and other users can be given the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please refer to the Security manual for further details.

FIIDs for LNET
Allows you to control user FIID access.
Available FIIDs Financial institution IDs available for access. You can highlight FIIDs from this list and move them to the Selected FIIDs list.

Note: This list is limited to the FIIDs specified on your own user record.

Selected FIIDs

The user is granted access to all the FIIDs within the Selected FIIDs list.

To remove access from the user record, highlight FIIDs in this list and move them back to the Available FIIDs list.

Back to Top

OrgIDs
Allows you to control user OrgID access.
Available OrgIDs OrgIDs available for access. You can highlight OrgIDs from this list and move them to the Selected OrgIDs list.

Note: This list is limited to the OrgIDs specified on your own user record.

Selected OrgIDs

The user is granted access to all the OrgIDs within the Selected OrgIDs list.

To remove access, highlight OrgIDs in this list and move them back to the Available OrgIDs list.
