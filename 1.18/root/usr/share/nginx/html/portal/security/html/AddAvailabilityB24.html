User Administration - Add Availability

Access to these applications can be further controlled by time. This page allows you to specify time restrictions for the new user. They are granted access only during the allotted time frame (U. S. Pacific Time).

Key Information
Auto-filled with data that identifies the selected user. This information cannot be changed.
First Name First name of the user whose record is being created.
Last Name Last name of the user whose record is being created.
Institution ID Six-digit identifier assigned by EFT for the selected institution.

Availability (Pacific Time)
Allows access time to be specified on the new user's security record.
Start Time Time of day when the user can begin to access the applications controlled by User Administration. Time format is military time and is defaulted to 00:00.
End Time Time of day after which the user can no longer access the applications controlled by User Administration. Time format is military time and is defaulted to 23:59.

Buttons
Back The Back button takes you to the previous page.

Clicking Back from the Add Availability page takes you to the Add Special Roles page.

Continue The Continue button temporarily stores the access times specified on the new user's security record and takes you to the next page.

Clicking Continue from the Add Availability page takes you to the Add Tokens page.

Cancel The Cancel button discontinues the add user process and returns you to the Choose User page.

Clicking Cancel from the Add Availability page displays the following confirmation message:
Click OK to exit Add User.
