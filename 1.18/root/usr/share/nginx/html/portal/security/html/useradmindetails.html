User Administration - User Details can be accessed as follows:
  1. Click the Add button on User Administration - Choose User.

    2. - or -
  2. Click the Add New User button on User Administration - Confirmation.

    3. - or -
  3. Click the Copy User button on User Administration - Choose User.

    4. - or -
  4. Click the Back button on User Administration - Logos.

    5. - or -
  5. Click Details on the User Record tab within User Administration - Navigation.

User Administration - User Details allows you to fill in basic information related to the user.

User Administration - User Details contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user whose record is being accessed or created.
  • Last Name – Last name of the user whose record is being accessed or created.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.
Note: This section displays only when an existing user record is being accessed.

User Information (contains view-only and modify fields)

Enter user information in the following fields:

  • First Name – First name of the user whose record is being accessed or created.
  • Last Name – Last name of the user whose record is being accessed or created.
  • Logo – Four-character alphanumeric client identifier.

User Address (contains modify fields)

  • Set Same as Yourself – Selecting this option prefills User Address fields with the address as defined on the security record of the administrator creating this record.
    Note: This field displays only when you access this page from the Add User button on User Administration - Choose User.
  • Institution – Name of the institution where the user is located.
  • Address – Address of the institution.
  • Country – Country where the institution is located.
  • City – City where the institution is located.
  • State – State where the institution is located.
  • ZIP – ZIP code where the institution is located.
  • Phone – Phone number of the user.
  • Ext – Extension of the user.
  • Fax – Fax number of the user.
  • Email – E-mail address of the user.

Buttons, links, and icons contained in User Administration - User Details:

Buttons
Continue – Allows you to continue to add the user record.

Cancel – Returns you to User Administration - Choose User or User Administration - User Record without saving changes.

OK – Allows you to temporarily save the modifications made to the page. Click the Save or Update button on User Administration - Navigation to permanently save your changes.

Note: The Continue button displays when pages are accessed from the Add button on User Administration - Choose User. Otherwise, the OK button displays.
