User Administration - Logos can be accessed as follows:
  1. Click the Continue button on User Administration - User Details.

    2. - or -
  2. Click the Back button on User Administration - Applications.

    3. - or -
  3. Click Logos on the User Record tab within User Administration - Navigation.

User Administration - Logos allows you to specify logos that the user may access via CWSi.

User Administration - Logos contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user whose record is being accessed or created.
  • Last Name – Last name of the user whose record is being accessed or created.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.
Note: The user ID displays only when an existing user record is being accessed.

Logos (contains modify fields)

  • Available Logos – Logos to which the user may have access.
  • Selected Logos – Logos to which the user is granted access.

Buttons, links, and icons contained in User Administration - Logos:

Buttons
>> – Moves one or more logos from Available Logos to Selected Logos.

All >> – Moves all logos from Available Logos to Selected Logos.

<< – Moves one or more logos from Selected Logos to Available Logos.

All << – Moves all logos from Selected Logos to Available Logos.

Back – Takes you back to the previous page.

Continue – Allows you to continue to add the user record.

Cancel – Returns you to User Administration - Choose User or User Administration - Navigation without saving changes.

OK – Allows you to temporarily save the modifications made to the page. Click the Save or Update button on User Administration - Navigation to permanently save your changes.

Note: The Back and Continue buttons display when pages are accessed from the Add button on User Administration - Choose User. Otherwise, the OK button displays.
