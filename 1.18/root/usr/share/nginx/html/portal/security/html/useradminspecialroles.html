User Administration - Special Roles can be accessed as follows:
  1. Click the Continue button on User Administration - Access Rights.

    2. - or -
  2. Click the Back button on User Administration - Availability.

    3. - or -
  3. Click Special Roles on the User Record tab within User Administration - Navigation.

User Administration - Special Roles allows you to specify special roles for the user being added on Client-Central.

User Administration - Special Roles contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user.
  • Last Name – Last name of the user.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.

Note: The user ID displays only when an existing user record is being accessed.

Special Roles (contains modify fields)

  • Available Roles – Special roles to which the user may be granted access.
  • Selected Roles – Special roles to which the user is granted access.

Buttons, links, and icons contained in User Administration - Special Roles:

Buttons
>> – Moves one or more special roles from Available Roles to Selected Roles.

All >> – Moves all special roles from Available Roles to Selected Roles.

<< – Moves one or more special roles from Selected Roles to Available Roles.

All << – Moves all special roles from Selected Roles to Available Roles.

Back – Takes you back to the previous page.

Continue – Allows you to continue adding the user record on User Administration - Availability.

Cancel – Takes you to User Administration - Choose User or User Administration - Navigation without saving changes.

OK – Allows you to temporarily save the modifications made to the page. Click the Save or Update button on User Administration - Navigation to permanently save your changes.

Note: The Back and Continue buttons display when pages are accessed from the Add button on User Administration - Choose User. Otherwise, the OK button displays.
