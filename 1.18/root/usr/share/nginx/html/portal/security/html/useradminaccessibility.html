User Administration - Availability can be accessed as follows:
  1. Click the Continue button on User Administration - Special Roles.

    2. - or -
  2. Click the Back button on User Administration - Tokens.

    3. - or -
  3. Click Availability on the User Record tab within User Administration - Navigation.

Note: This page is accessible by clicking the Continue button on User Administration - Access Rights in case the User Administration - Special Roles is not applicable.

User Administration - Availability allows you to specify access time restrictions for the user.

User Administration - Availability contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user.
  • Last Name – Last name of the user.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.

Note: The user ID displays only when an existing user record is being accessed.

Availability (contains modify fields)

  • Start Time – Time of day when the user can begin to access Client-Central.
  • End Time – Time of day after which the user can no longer access Client-Central.

Note: Time format is military time.

Buttons, links, and icons contained in User Administration - Availability:

Buttons
Back – Takes you back to the previous page.

Continue – Allows you to continue adding the user record on User Administration - Tokens.

Cancel – Takes you to User Administration - Choose User or User Administration - Navigation without saving changes.

OK – Allows you to temporarily save the modifications made to the page. Click the Save or Update button on User Administration - Navigation to permanently save your changes.

Note: The Back and Continue buttons display when pages are accessed from the Add button on User Administration - Choose User. Otherwise, the OK button displays.
