Buttons

When Client ID radio button is selected:

View

 When selected, displays the Results section for the selected institution which includes the Institution ID value, Soft Token Yes/No drop-down and the IP Whitelisting Yes/No drop-down.
Add

When selected, adds the Logo entered, displays the success message Institution has been added, and the Results values for the selected institution which includes the Institution ID field, Soft Token Yes/No drop-down, and the IP Whitelisting Yes/No drop-down.

When Web Service Digital Certificate ID radio button is selected:

View

 When selected, displays the CardValet AppToken for the Digital Certificate ID entered. See CardValet AppToken for more information.

If no AppToken is associated with the Digital Certificate entered, the following message will display: Digital Certificate ID retrieved successfully. No AppToken is associated with Digital Certificate.
Add

If selected, the entered Digital Certificate ID is added to LDAP and the following message displays: Digital Certification added successfully.

If the Digital Certificate already exists the following message displays: Digital Certificate retrieved successfully.
Delete

If selected, the entered Digital Certificate ID is added to LDAP and the following message displays: Digital Certification added successfully.

If the Digital Certificate already exists the following message displays: Digital Certificate retrieved successfully.
Add CardValet

If selected, the entered Digital Certificate ID is added to LDAP and the following message displays: Digital Certification added successfully.

If the Digital Certificate already exists the following message displays: Digital Certificate retrieved successfully.

Results

When Client ID radio button is selected:

Institution ID View only field, redisplays Logo entered.
Soft Token Yes/No drop-down. Not currently supported. Will control the ability for an institution to add soft tokens.

This field should always be set to No.

IP Whitelisting Yes/No drop-down. Controls the display of the IP Whitelisting tab in User Administration to external administrators.

Yes IP Whitelisting tab will display for all client users who have access to User Administration.
No IP Whitelisting tab will not display for the client's users.

Note: The IP Whitelisting tab will always display for Internal users.

Updated button When clicked, will apply any changes made to the page and display success message: The institution has been updated.
Delete button When clicked, a confirmation pop up box displays. Select OK to delete. The following success message displays: Institution ID deleted successfully.

CardValet AppToken
This section displays when the entered Digital Certificate ID is associated with an AppToken
Add AppToken checkbox When not checked, the existing AppToken UserName and Password can be modified.

When checked, credentials for the new AppToken can be entered.
AppToken Display Name When the Add AppToken checkbox is not checked, this drop down menu displays all available AppToken mappings.

When the Add AppToken checkbox is checked, a selection may be made from the list.

AppToken UserName This field is read-only and will display the selected AppToken's username.
AppToken Password When the Add AppToken checkbox is not checked, the existing password can be edited.

Note: Password cannot be less than 8 characters.

When the Add AppToken checkbox is checked, enter the desired password for the new AppToken.

Re-Enter Password Re-enter the desired password.
Assign Credentials Assigns the entered credentials, UserName and Password, to the Digital Certificate.
Delete If selected, deletes the Digital Certificate ID entered. A confirmation message will display. Select OK to delete.
