User Administration - Applications can be accessed as follows:
  1. Click the Continue button on User Administration - Logos.

    2. - or -
  2. Click the Back button on User Administration - Access Rights.

    3. - or -
  3. Click Applications on the User Record tab within User Administration - Navigation.

User Administration - Applications allows you to specify application access for the user.

User Administration - Applications contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user.
  • Last Name – Last name of the user.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.

Note: The user ID displays only when an existing user record is being accessed.

Applications (contains modify fields)

  • Available Applications – Applications to which the user may have access.
  • Selected Applications – Applications to which the user is granted access.

Buttons, links, and icons contained in User Administration - Applications:

Buttons
>> – Moves one or more applications from Available Applications to Selected Applications.

All >> – Moves all applications from Available Applications to Selected Applications.

<< – Moves one or more applications from Selected Applications to Available Applications.

All << – Moves all applications from Selected Applications to Available Applications.

Back – Takes you back to the previous page.

Continue – Allows you to continue to add or modify the user record on User Administration - Access Rights.

Cancel – Takes you to User Administration - Choose User or User Administration - Navigation without saving changes.

Note: The Back button displays only when pages are accessed from the Add button on User Administration - Choose User.
