User Administration - Reports can be accessed as follows:
  1. Click the Reports tab within User Administration - Choose User.

User Administration - Reports contains service request information for all open requests, as well as for requests completed in the past 30 calendar days.

User Administration - Reports contains the following information:

Request Report (contains view-only fields)

The following information displays in table format:

  • Request ID – A sequential number allocated to every request.
  • User ID – Client-Central user ID of the user.
  • User Name – First and last name of the user.
  • Request Type – Type of service requested.
  • Request Date – Date of the service request.
  • Status – Current status of the request.
  • Deliver To – Name of the person designated to receive the token.
  • Address – Destination street address of the shipment.
  • City, State ZIP – Destination city, state, and ZIP code of the shipment.
