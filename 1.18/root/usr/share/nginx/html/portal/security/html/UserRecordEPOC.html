User Administration—User Record

This page is your navigation page, allowing you access to all available information for the selected user. To view or change information, select the relevant page from the second row of menu items displayed. The menu items available include Details, Institution IDs, Applications, Special Roles, Availability, and Token.

Note: The Access Rights page is accessible from the Applications menu item.

You are returned to this page each time you click OK from the page you selected. You can continue making changes to other pages as needed. All of your changes will be held until you check the Approval box and click Update. At that time, your changes are applied and the Confirmation page displays allowing you to view the updated user record. You can also delete security records from this page.

Please refer to the Security manual for further details.

Buttons
Update Applies all updates made on the various tabs for the selected user ID and takes you to a confirmation page.
Disable/Enable Disable—Allows you to disable the selected user record preventing them from logging in with their token.
Enable—Available when a user record is disabled, allowing you to reenable the record.
Delete Deletes the user security record, returns you to the Choose User page, and displays a confirmation message. The user record is permanently deleted from Client-Central.
Cancel Discards any changes you have made on the record and returns you to the Choose User page.
