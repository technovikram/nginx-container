User Administration - Add Access Rights

This page allows you to set access rights for applications available to the new user. Some applications, like CVi - Reports, do not require access rights in User Administration and are not displayed on this page. Other applications, like CWSi - Card Management, have access rights split out by page and all pages are listed here. Some applications, like CWSi - User Administration, have just one set of access rights for the entire application.

Please be aware that changes made on the Applications page are reflected on this page. If an application is removed, it no longer shows up on this page. If an application is added, access rights need to be specified. Since Applications and Access Rights are dependent on each other, they are always found together.

Note: Every institution has one designated primary security administrator. Maximum privileges applicable to the institution are defined on this user's security record. Other administrators and users can be given the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please refer to the User Administration User Manual for further details.

Buttons
Select All This button automatically checks all allowable access rights for the applications listed.
Clear This button automatically unchecks all allowable access rights.
Reset This button resets access rights to the original values existing at the time the page was accessed.
Read Only This button automatically checks read-only access rights for the identified pages within each application.
Back The Back button temporarily stores the access right information specified on the new user's security record and takes you to the Add Applications page.
Continue The Continue button temporarily stores the access right information specified on the new user's security record and takes you to the next page.

Clicking Continue from the Add Access Rights page takes you to the Add Special Roles page or to the Add Availability page. The Add Special Roles page displays only if you have at least one special role defined on your security record.

Cancel The Cancel button discontinues the add user process and returns you to the Choose User page.

Clicking Cancel from the Add Access Rights page displays the following message:
Click OK to exit Add User.
