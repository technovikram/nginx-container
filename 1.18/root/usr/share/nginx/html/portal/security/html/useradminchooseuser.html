User Administration - Choose User can be accessed as follows:
  1. Click the User Administration tab.
  2. Click User Administration on the gray navigation column.
Note: If you are already in the User Administration application, click the Choose User tab to go to the User Administration - Choose User page.

User Administration - Choose User allows you to search for security records of Client-Central users associated with your institution based on the information you provide. It also allows you to view, manage, or add user security records.

User Administration - Choose User contains the following information:

Search (contains modify fields)

Enter specific information in the following fields to run a search:

  • Logo – Allows you to search for a user security record based on client logo.
  • User ID – Allows you to search for a user security record based on Client-Central user ID.
  • First Name – Allows you to search for a user security record based on first name.
  • Last Name – Allows you to search for a user security record based on last name.

Note: You may use any combination of fields to enter search criteria.

Results (contains view-only fields)

Results display in table format. Each row represents a user security record. You can select a particular record from the list to view details or to manage the record.

Buttons, links, and icons contained in User Administration - Choose User:

Buttons
Add User – Allows you to add users to Client-Central.

Search – Allows you to search for user security records on Client-Central.

Clear – Allows you to clear all editable fields in the Search section.

View User – Allows you to view the details of a selected user security record from the Results table.

Manage User – Allows you to edit or delete the selected user security record.

Copy User – Allows you to add a new user by copying the rights and settings from an existing user's record.

Links
Print – Allows you to print the contents of the Results table.

Icons
None
