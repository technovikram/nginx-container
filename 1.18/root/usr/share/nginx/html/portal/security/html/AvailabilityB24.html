User Administration - Availability

Access to these applications can be further controlled by time. This page allows you to specify time restrictions for each user. They are granted access only during the allotted timeframe (U.S. Pacific Time).

Institution ID Six-digit identifier assigned by EFT for the selected institution.

Availability (Pacific Time)
You can edit the access time on the user's record.
Start Time Time of day when the user can begin to access the applications controlled by User Administration. Time format is military time (00:00).
End Time Time of day after which the user can no longer access the applications controlled by User Administration. Time format is military time (23:59).

Buttons
OK The OK button temporarily stores the information entered and takes you to the User Record page. You can then choose to save your changes by selecting Update or you can make additional changes on any of the other tabs available. All changes are held until you select Update.
Cancel The Cancel button discards your changes and returns you to the User Record page.
