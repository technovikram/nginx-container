User Administration—User Record

This page displays when you select the Manage User button. It is your "navigation" page allowing you access to all available information for the selected user. To view or change information, select the relevant page from the second row of tabs displayed. The tabs available include Details, Institution IDs, Applications, Special Roles, Availability, and Tokens.

Note: The Access Rights page is accessible in the Applications tab.

You are returned to this page each time you click OK from one of these pages. You can make changes to multiple pages. They are held and are not applied until you check the Approval box and click Update. At that time, your changes are applied and the Confirmation page displays allowing you to view the updated user record.

Please refer to the Security manual for further details. This page also allows you to delete a user security record if you have delete access and select the Delete button.

Buttons
Update Applies all updates made on the various tabs for the selected user ID and takes you to a confirmation page.
Disable/Enable Disable—Allows you to disable the selected user record preventing them from logging in with their token.
Enable—Available when a user record is disabled allowing you to reenable the record.
Delete Deletes the user security record, returns you to the Choose User page, and displays a confirmation message. The user record is permanently deleted.
Cancel Discards any changes you have made on the record and returns you to the Choose User page.
