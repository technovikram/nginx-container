User Administration - Choose User

The Choose User page gives you access to the User Administration application. Only the tabs and buttons available to your user ID display on this page.

If you do not have security access to one of these options you will not see it. This help page reviews all possible options on the Choose User page and may contain items you do not see.

The tabs at the top of the page–Choose User, Reports, and Defaults–allow you to view and maintain your institution's user security records, view reports, and set default information for the application.

The Add User button takes you into the Add Wizard which walks you through each page possible for adding a user security record.

In the Search section, you can enter information into one or more of the available fields to search for user records. The Results section displays relevant data based on the search criteria entered. The system retains the search criteria and results on this page when you return to it from within a user security record.

Please use the buttons and tabs provided in this application for navigation. The Enter key is not a valid option and may give you unexpected results.

Search
This section allows you to search for user security records. You may use a single field or any combination of the available fields for your search.
Institution ID Allows you to search for a user security record based on institution ID.
PSA (check box) Allows you to search for the Primary Security Administrator designated for your institution.
User ID Allows you to search for a user security record based on Client-Central user ID.

Note: Partial data can be entered.

First Name Allows you to search for a user security record based on first name.

Note: Partial data can be entered.

Last Name Allows you to search for a user security record based on last name.

Note: Partial data can be entered.

Results
This section displays a read-only result list, displayed in table format, based on the search criteria entered. Each row represents a user security record. You can select a particular record from the list to view details or to manage the record. The columns displayed as part of the result set include the following:
User ID User's Client-Central user ID.
First Name First name of the user.
Last Name Last name of the user.
Disabled A user security record is disabled after 90 days of inactivity in order to comply with security and audit requirements. If Disabled displays Yes, the user cannot access Client-Central.
Days Inactive Indicates the number of days of inactivity.
Create Date/Time Indicates the date and time when the particular user security record was initially added.
Last Login Date/Time Indicates the date and time when the user last logged onto Client-Central.

Buttons
Add User Selecting the Add User button begins the Add Wizard. The Add Wizard guides you through a series of pages enabling you to enter required information on the security record of the new user. Add Cardholder Details is the first page of the Add Wizard.

Note: This button displays only if you have Add Cardholder security access.

Search Allows you to search for user security records on Client-Central. A result list displays on the Cardholder Selection page, based on the search criteria entered. You can choose to search on one or more of the criteria as detailed above.
Clear The Clear button removes all results and search criteria from the page.
View User For all search options, this button takes you to a read-only page that displays the security record for the row highlighted on the results list after a search has been run and the results displayed.
Manage User For all search options, this button takes you into the security record for the row highlighted on the results list after a search has been run and the results displayed. You can then opt to modify or delete the selected user security record.
Copy User Allows you to copy the rights and settings from an existing user's record when adding a new user. Select the record to be copied by highlighting a row on the results list after a search has been run and click the Copy User button. This process takes you through all of the relevant pages just like Add User but there is one difference—most of the information is copied from the selected record. This information can be modified as needed for the new user being added.
User Download This button creates a PDF that contains user summary information for all users contained within the search results list. It can be viewed, printed, or downloaded.

If the search results list exceeds 100 users, a Starting Row # field appears beside the User Download button. Since the PDF can only contain user summary information for up to 100 users, this field allows you to designate which group of 100 will be contained within each PDF. To generate a PDF for the first 100 user summary records, enter the starting row value of 1 and click the User Download button. Once the PDF for that group has been downloaded or printed, enter a starting row value of 101 to generate a new PDF for the next group and so on until you have captured all records within the search results list.
