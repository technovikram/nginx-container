User Administration - Defaults can be accessed as follows:
  1. Click the Defaults tab within User Administration - Choose User.

User Administration - Defaults allows you to specify default shipping address information.

User Administration - Defaults contains the following information:

Shipping Address (contains modify fields)

  • Ship To – The default shipping address option that should be preselected.

Default Address (contains view-only and modify fields)

  • Institution – Name of the institution to which the token should be shipped.
  • Deliver To – Name of the person designated to receive the token.
  • Address – Shipping address.
  • Country – Destination country of the shipment.
  • City – Destination city of the shipment.
  • State – Destination state of the shipment.
  • ZIP – Destination ZIP code of the shipment.
  • Phone – Phone number of the person designated to receive the token.
  • Ext – Extension of the person designated to receive the token.
  • Fax – Fax number of the person designated to receive the token.
  • Email – E-mail address of the person designated to receive the token.

Buttons, links, and icons contained in User Administration - Defaults:

Buttons
Save Defaults – Saves the defaults and takes you to User Administration - Choose User.

Cancel – Takes you to User Administration - Choose User without saving changes.
