User Administration - Choose User

The Choose User page gives you access to the User Administration application. Only the tabs and buttons available to your user ID display on this page. If you do not have security access to one of these options you do not see it. This help page reviews all possible options on the Choose User page and may contain items you do not see.

The tabs at the top of the page—Choose User, Reports, and Defaults—allow you to view and maintain your institution's user security records, view reports, and set default information for the application.

The Add User button takes you into the Add Wizard which walks you through each page possible for adding a user security record.

In the Search section, you can enter information into one or more of the available fields to search for user records. The Results section displays relevant data based on the search criteria entered. The system retains the search criteria and results on this page when you return to it from within a user security record.

Please use the buttons and tabs provided in this application for navigation. The Enter key is not a valid option and may give you unexpected results.

Search
This section allows you to search for user security records. You may use a single field or any combination of the available fields for your search.
Institution ID Displays the financial institution ID associated with your token ID.
PSA (check box) Allows you to search for the Primary Security Administrator designated for your institution.
User ID Allows you to search for a user security record by user ID.

Note: Partial data cannot be entered but the field can be left blank.

First Name Allows you to search for a user security record based on first name.

Note: Partial data cannot be entered but the field can be left blank.

Last Name Allows you to search for a user security record based on last name.

Note: Partial data cannot be entered but the field can be left blank.

Results
This section displays a read-only results list in table format based on the search criteria entered. Each row represents a user security record. You can select a particular record from the list to view details or to manage the record. A records found total displays at the bottom of the results list. A scroll bar appears when the results list contains more than 11 records.
User ID User ID of the user security record.
First Name First name associated with the user ID.
Last Name Last name associated with the user ID.
Disabled Status of the user ID. A user security record is disabled after 90 days of inactivity to comply with security and audit requirements. If Disabled displays Yes, the user is not allowed to access any of their applications.
Create Date/Time Date and time when the user security record was initially added.

Buttons
Add User Selecting the Add User button begins the Add Wizard. The Add Wizard guides you through the pages and information required to establish a new user security record. Add User Details is the first page of the Add Wizard.

Note: This button displays only if you have add access to the User Administration application.

Search Allows you to search for user security records in User Administration. A results list displays based on the search criteria entered. See the Search section above for more information about the available search criteria.
Clear Removes all results and search criteria from the page.
View User Displays after the results of a search are returned. Highlight the user ID you want to view and click this button. You are taken to the read-only User Summary page for the selected user.
Manage User Displays after the results of a search are returned. Highlight the user ID you want to manage (update or delete) and click this button. You are taken to the User Record (navigation) page which allows access to all available tabs for the selected user—User Details, FIIDs, Applications, Special Roles, Availability, and Tokens—as well as the Update and Delete buttons.
Copy User Displays after the results of a search are returned. Highlight the user ID you want to copy and click this button. You are taken to the Add Wizard but much of the relevant information is already filled in for you based on the selected user ID. This information can be modified as needed for the new user being added.
User Download This button creates a PDF that contains user summary information for all users contained within the search results list. It can be viewed, printed, or downloaded.

If the search results list exceeds 100 users, a Starting Row # field appears beside the User Download button. Since the PDF can only contain user summary information for up to 100 users, this field allows you to designate which group of 100 will be contained within each PDF. To generate a PDF for the first 100 user summary records, enter the starting row value of 1 and click the User Download button. Once the PDF for that group has been downloaded or printed, enter a starting row value of 101 to generate a new PDF for the next group and so on until you have captured all records within the search results list.
