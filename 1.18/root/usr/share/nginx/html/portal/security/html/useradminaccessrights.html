User Administration - Access Rights can be accessed as follows:
  1. Click the Continue button on User Administration - Applications.

    2. - or -
  2. Click the Back button on User Administration - Special Roles.
    Note: This page is accessible by clicking the Back button on User Administration - Availability in case User Administration - Special Roles is not applicable.

User Administration - Access Rights allows you to specify access rights on the applications granted to the user.

User Administration - Access Rights contains the following information:

Key Information (contains view-only fields)

  • First Name – First name of the user.
  • Last Name – Last name of the user.
  • Logo – Four-character alphanumeric client identifier.
  • User ID – Client-Central user ID of the user.

Note: The user ID displays only when an existing user record is being accessed.

Access Rights (contains view-only and modify fields)

You can grant and revoke read, write, modify, and delete access within each application.

Buttons, links, and icons contained in User Administration - Access Rights:

Buttons
Select All – Grants all allowable access rights.

Clear – Revokes all access rights.

Reset – Resets access rights to the original values existing at the time when the page was accessed.

Back – Takes you back to the previous page.

Continue – Allows you to continue adding the user record on User Administration - Special Roles or User Administration - Availability.

Cancel – Takes you to User Administration - Choose User or User Administration - Navigation without saving changes.

OK – Allows you to temporarily save the modifications made to the page. Click the Save or Update button on User Administration - Navigation to permanently save your changes.

Note: The Back and Continue buttons display when pages are accessed from the Add button on User Administration - Choose User. Otherwise, the OK button displays.
