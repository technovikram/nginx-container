User Administration—Institutions

Client-Central provides the user with access to cardholder information relating to one or more client institutions, termed logos for debit clients and OrgIDs for Full Service Credit clients. This page allows you to grant or revoke user access to specific logos and/or OrgIDs. The user has access to the logos within the Selected Logos and/or Selected OrgID list.

When the user has the EPOC System checked on the User Details page, the Logos/Client IDs section displays. When the Full Service Credit System is checked, the OrgIDs section displays. 

Note: Every institution has one designated primary security administrator (PSA). Maximum privileges applicable to the institution are defined on the PSA's security record. Other administrators and other users can be given the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please refer to the Security manual for further details.

Logos/Client IDs
Allows you to control user logo access.
Available Logos Logos available for access. You can highlight logos from this list and move them to the Selected Logos list.

Note: This list is limited to the logos specified on your own user record.

Selected Logos

The user is granted access to all the logos within the Selected Logos list.

To remove access, highlight logos from this list and move them to the Available Logos list.

Back to Top

OrgIDs
Allows you to control user OrgID access.
Available OrgIDs

OrgIDs available for access. You can highlight OrgIDs from this list and move them to the Selected OrgIDs list.

Note: This list is limited to the OrgIDs specified on your own user record.

Selected OrgIDs

The user is granted access to all the OrgIDs within the Selected OrgIDs list.

To move access, highlight OrgIDs from this list and move them to the Available OrgIDs list.
