User Administration—Add Institutions

Client-Central provides the user with access to cardholder information relating to one or more client institutions, called logos for debit clients and OrgIDs for Full Service Credit clients. This page allows you to grant the new user access to specific logos and/or OrgIDs.

When the user has the EPOC System checked on the User Details page, the Logos/Client IDs section displays. When the Full Service Credit System is checked, the OrgIDs section displays.

The Add Institutions page displays for all users and shows all of your financial institution IDs. Many users have access to only one institution and, if that is the case, it is the only Logo/Client ID or OrgID that displays on this page. If you have access to multiple institutions, they all display in the Available list and you are allowed to determine which Logos and/or OrgIDs the new user should have access to by moving them to the Selected list.

Note: Every institution has one designated primary security administrator. Maximum privileges applicable to the institution are defined on this user's security record. Other administrators and other users can be given the same or lesser privileges. When adding or maintaining a user record, you are not able to give more access than you have. Please see the Security manual for further details.

Logos/Client IDs
Allows you to specify logos that can be accessed by the new user.
Available Logos Logos available for access. You can highlight logos in this list and move them to the Selected Logos list.

Note: This list is limited to the logos specified on your own user record.
Selected Logos

The user is granted access to all the logos within the Selected Logos list.

To remove access from the user record, you can highlight logos in this list and move them to the Available Logos list.

Back to Top

OrgIDs
Allows you to specify OrgIDs that can be accessed by the new user.
Available OrgIDs

OrgIDs available for access. You can highlight OrgIDs in this list and move them to the Selected OrgIDs list.

Note: This list is limited to the OrgIDs specified on your own user record.

Selected OrgIDs

The user is granted access to all the OrgIDs within the Selected OrgIDs list.

To remove access from the user record, you can highlight OrgIDs in this list and move them to the Available OrgIDs list.

Buttons
>> Moves the highlighted items from the Available list to the Selected list.
All >> Moves all items in the section from the Available list to the Selected list.
<< Moves the highlighted items from the Selected list to the Available list.
All << Moves all items in the section from the Selected list to the Available list.
Back Temporarily stores the data entered on this page for the new user's security record and takes you to the previous page.

Clicking Back from the Add Institutions page takes you to the Add User Details page.

Continue Temporarily stores the data entered on this page for the new user's security record and takes you to the next page.

Clicking Continue from the Add Institutions page takes you to the Add Applications page.

Cancel Discontinues the add user process and returns you to the Choose User page.

Clicking Cancel from the Add Institutions page displays the following confirmation message:
To cancel this Add, click OK.
To return to the Add, click Cancel.
