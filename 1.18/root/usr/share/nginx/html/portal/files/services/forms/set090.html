Visa Preauthorized Payment Cancellation
  The purpose of this form is to cancel preauthorized electronic funds transfers, such as automatic bill payments and installment payments. Please note: transactions that have already received an authorization approval must be allowed to clear and settle (PPCS will not stop these).

You can use this form to request one of the following:
  • Stop payment order - stops a single specific preauthorized payment to a single merchant.
  • Revocation of authorization order - stops all future preauthorized payments to a particular merchant.
  • Revocation of all authorizations order - stops all bill payment transactions for a specific closed account.

Merchant Name
  We suggest entering the merchant’s name exactly as it appears on the authorization record (PAD) in the Transaction Journal (TJ). If the merchant name differs between the authorization record (110 message) and the clearing record (220 message), include both names to prevent payment. Note: Although EFT only charges one fee per PPCS request, Visa charges a fee for each merchant name submitted (charged via Visa Integrated Billing).
